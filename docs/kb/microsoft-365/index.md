# Microsoft 365 Overview

Microsoft 365 is a comprehensive suite of productivity and collaboration tools that includes Office applications, cloud services, and enterprise security features.

## What is Microsoft 365?

Microsoft 365 combines familiar Office applications with cloud-based services to help organizations work more efficiently and securely. It includes email, file storage, collaboration tools, and administrative controls all in one integrated platform.

## Key Features

- **Professional Email** - Outlook with Exchange Online
- **Cloud Storage** - OneDrive and SharePoint for file storage and sharing
- **Office Applications** - Word, Excel, PowerPoint, and more
- **Collaboration** - Microsoft Teams for chat, meetings, and collaboration
- **Security** - Microsoft Entra (formerly Azure AD) for identity and access management
- **Admin Controls** - Centralized administration and security management
- **Mobile Access** - Work from anywhere with mobile apps
