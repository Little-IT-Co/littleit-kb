# Adding Users

Adding users to your Microsoft 365 organization allows them to access services and applications included in your subscription.

## Overview

User accounts in Microsoft 365 provide access to email, Office applications, cloud storage, and other services based on the licenses assigned to them.

## Adding Users

### Method 1: Microsoft 365 Admin Center

1. Sign in to the [Microsoft 365 admin center](https://admin.microsoft.com)
2. Navigate to **Users** → **Active users**
3. Click **Add a user**
4. Fill in the user details:
   - First name and last name
   - Display name
   - Username (will be used for email address)
   - Domain selection
5. Assign licenses to the user
6. Set the user's password (or allow auto-generated password)
7. Choose whether to send password to user
8. Click **Finish adding**

### Method 2: Bulk Import

For adding multiple users at once:

1. Go to **Users** → **Active users**
2. Click **Add multiple users**
3. Download the CSV template
4. Fill in user information in the CSV file
5. Upload the completed CSV file
6. Assign licenses
7. Review and create users

## User Details

### Required Information
- **First Name** - User's first name
- **Last Name** - User's last name
- **Display Name** - Name shown in directory and email
- **Username** - Email username (before @domain.com)
- **Domain** - Select from available domains

### Optional Information
- **Job Title** - User's job title
- **Department** - User's department
- **Office Location** - Physical office location
- **Phone Number** - Contact phone number

## License Assignment

### Assigning Licenses
1. Select the user(s) during creation
2. Choose from available licenses
3. Or assign licenses later from user properties

### License Types
- **Microsoft 365 Business Basic** - Web and mobile apps
- **Microsoft 365 Business Standard** - Desktop apps + web apps
- **Microsoft 365 Business Premium** - Includes advanced security
- **Enterprise Plans** - E3, E5, etc.

## User Roles

### Admin Roles
- **Global Administrator** - Full access to all features
- **User Administrator** - Manage users and licenses
- **Exchange Administrator** - Manage Exchange settings
- **SharePoint Administrator** - Manage SharePoint

### User Roles
- **User** - Standard user with no admin rights
- **Guest User** - External user with limited access

## Password Management

### Setting Initial Password
- **Auto-generated** - System generates secure password
- **Custom Password** - Administrator sets password
- **Force Password Change** - Require password change on first login

### Password Policies
- Minimum length requirements
- Complexity requirements
- Password expiration policies
- Multi-factor authentication

## User Communication

### Welcome Email
- Option to send welcome email with:
  - Username
  - Temporary password
  - Login instructions
  - Links to setup resources

### Manual Communication
- Provide credentials securely
- Include setup instructions
- Share support contact information

## Best Practices

- **Use Consistent Naming** - Follow a naming convention for usernames
- **Assign Appropriate Licenses** - Only assign licenses users need
- **Set Strong Passwords** - Use strong, unique passwords
- **Enable MFA** - Enable multi-factor authentication
- **Document Changes** - Keep records of user additions
- **Train Users** - Provide onboarding resources

## Editing Users

### Changing User Information
1. Select the user in Active users
2. Click **Edit** or click on the user's name
3. Update information as needed
4. Save changes

### Common Changes
- Update contact information
- Change display name
- Add or remove licenses
- Reset password
- Change user role

## Removing Users

### Steps to Remove
1. Select the user(s) to remove
2. Click **Delete user**
3. Choose to:
   - Delete immediately
   - Block sign-in but retain data
   - Transfer data to another user
