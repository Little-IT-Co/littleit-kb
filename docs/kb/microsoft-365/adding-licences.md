# Adding Licences

Managing licenses in Microsoft 365 ensures users have access to the services and features they need.

## Overview

Licenses determine which Microsoft 365 services and features each user can access. Proper license management is essential for cost control and ensuring users have appropriate access.

## Understanding Licenses

### License Types

#### Microsoft 365 Business Plans
- **Business Basic** - Web and mobile Office apps, email, Teams
- **Business Standard** - Everything in Basic + desktop Office apps
- **Business Premium** - Everything in Standard + advanced security

#### Microsoft 365 Enterprise Plans
- **E3** - Full Office suite + advanced security and compliance
- **E5** - Everything in E3 + advanced analytics and security features

### What Licenses Include
- **Email** - Exchange Online mailbox
- **Office Apps** - Word, Excel, PowerPoint, etc.
- **Cloud Storage** - OneDrive storage
- **Teams** - Collaboration and meetings
- **Security Features** - Varies by plan
- **Compliance Tools** - Varies by plan

## Purchasing Licenses

### Adding Licenses to Existing Subscription

1. Sign in to [Microsoft 365 admin center](https://admin.microsoft.com)
2. Navigate to **Billing** → **Licenses**
3. Click **Add licenses** or **Buy licenses**
4. Select the license type and quantity
5. Review and complete purchase

### Adding Licenses During Setup

1. During initial setup, specify number of licenses needed
2. Licenses are added automatically to your subscription
3. Assign licenses to users as needed

## Assigning Licenses

### Assign to Individual User

1. Go to **Users** → **Active users**
2. Select the user
3. Click on **Licenses and apps**
4. Select the license(s) to assign
5. Save changes

### Assign to Multiple Users

1. Select multiple users in Active users
2. Click **Manage product licenses**
3. Choose license assignment
4. Select licenses to assign or remove
5. Apply changes

### Bulk License Assignment

1. Use PowerShell for bulk operations
2. Or use the admin center to select multiple users
3. Apply license changes in bulk

## License Management

### Viewing License Usage

1. Go to **Billing** → **Licenses**
2. See:
   - Total licenses purchased
   - Licenses assigned
   - Licenses available
   - License expiration dates

### Unassigning Licenses

1. Select user(s) in Active users
2. Click **Manage product licenses**
3. Deselect licenses to remove
4. Save changes

### License Reassignment

When removing licenses from users:
- Data is typically retained for 30 days
- Can reassign license to recover data
- Consider data export before removal

## License Best Practices

### Cost Management
- **Review Regularly** - Audit license usage quarterly
- **Remove Unused Licenses** - Unassign from inactive users
- **Right-Size Plans** - Use appropriate plan for each user's needs
- **Monitor Usage** - Track license utilization

### Compliance
- **Match Licenses to Needs** - Ensure users have appropriate access
- **Document Assignments** - Keep records of license assignments
- **Plan for Growth** - Budget for additional licenses as needed

### User Experience
- **Assign Promptly** - Assign licenses when users are added
- **Communicate Changes** - Notify users of license changes
- **Provide Training** - Help users understand available features

## License Add-ons

### Additional Services
Some services require add-on licenses:
- **Advanced Threat Protection**
- **Information Protection**
- **Audio Conferencing**
- **Phone System**

### Adding Add-ons
1. Purchase add-on licenses
2. Assign to users who need them
3. Configure service settings

## License Expiration

### Monitoring Expiration
- Check license expiration dates regularly
- Renew licenses before expiration
- Set up renewal reminders

### Expiration Process
- Users lose access when licenses expire
- Data is typically retained for 30 days
- Renew licenses to restore access

## Troubleshooting

### Common Issues
- **Can't Assign License** - Check available licenses
- **User Can't Access Service** - Verify license assignment
- **License Not Showing** - Wait for sync or refresh
- **Wrong Features Available** - Check license type
