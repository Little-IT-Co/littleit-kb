# Mailboxes

Managing mailboxes in Exchange Online involves understanding mailbox types, storage limits, and configuration options.

## Overview

Mailboxes in Exchange Online provide email storage and access for your Microsoft 365 users. Different mailbox types serve different purposes within your organization.

## Mailbox Types

### User Mailboxes
Standard mailboxes assigned to individual users for sending and receiving email.

### Shared Mailboxes
Mailboxes that multiple users can access for collaborative purposes (e.g., support@yourdomain.com).

### Resource Mailboxes
Special mailboxes for meeting rooms and equipment scheduling.

### Distribution Groups
Email groups that forward messages to multiple recipients.

## Mailbox Storage

### Storage Limits
- **User Mailboxes** - Typically 50GB or 100GB depending on license
- **Shared Mailboxes** - 50GB (free) or licensed capacity
- **Archive Mailboxes** - Additional storage for compliance and retention

### Managing Storage
- Monitor mailbox sizes in the Microsoft 365 admin center
- Enable archive mailboxes for additional storage
- Set up retention policies to manage old emails
- Use mailbox cleanup tools

## Mailbox Configuration

### Email Addresses
- **Primary Address** - Default email address (user@domain.com)
- **Aliases** - Additional email addresses pointing to the same mailbox
- **Multiple Domains** - Support for multiple email domains

### Mailbox Permissions
- **Full Access** - Complete access to another user's mailbox
- **Send As** - Send email on behalf of another user
- **Send on Behalf** - Send email as another user with indication
- **Folder Permissions** - Granular access to specific folders

### Automatic Replies
- Set up out-of-office messages
- Configure automatic forwarding rules
- Schedule automatic replies

## Shared Mailboxes

### Creating Shared Mailboxes
1. Go to Microsoft 365 admin center
2. Navigate to **Exchange** → **Recipients** → **Shared**
3. Click **Add shared mailbox**
4. Enter mailbox details and assign members

### Benefits
- No additional license required (up to 50GB)
- Multiple users can access the same mailbox
- Perfect for team email addresses
- Shared calendar and contacts

## Archive Mailboxes

### Enabling Archive
- Provides additional storage for email retention
- Can be enabled automatically with retention policies
- Accessible through Outlook and Outlook on the web

### Benefits
- Long-term email storage
- Compliance and legal requirements
- Automatic archiving based on policies
- Improved performance for primary mailbox

## Mailbox Policies

### Retention Policies
- Define how long emails are kept
- Automatically move items to archive
- Comply with legal and business requirements

### Mail Flow Rules
- Automatically forward emails
- Apply disclaimers
- Route emails based on conditions
- Filter spam and malicious content

## Best Practices

- **Regular Monitoring** - Monitor mailbox sizes regularly
- **Archive Old Email** - Use archive mailboxes for old emails
- **Clean Up** - Regularly delete unnecessary emails
- **Use Shared Mailboxes** - Use shared mailboxes for team addresses
- **Set Retention Policies** - Implement retention policies early
- **Permissions** - Grant only necessary permissions

## Troubleshooting

### Common Issues
- **Mailbox Full** - Enable archive or increase storage
- **Can't Access Mailbox** - Check permissions and licenses
- **Email Not Arriving** - Check mail flow rules and spam filters
- **Sync Issues** - Clear cache or reconfigure client
