# Google Site Kit

Google Site Kit is a WordPress plugin that provides official Google integrations for your website, including Analytics, Search Console, AdSense, and more.

## Overview

Google Site Kit brings together essential Google services into a single WordPress dashboard, making it easy to monitor your site's performance, search visibility, and monetization.

## Key Features

- **Google Analytics** - Track website traffic and user behavior
- **Google Search Console** - Monitor search performance and indexing
- **Google AdSense** - Manage ad monetization
- **PageSpeed Insights** - Measure site performance
- **Tag Manager** - Manage tracking codes
- **Unified Dashboard** - All Google services in one place

## Installation

### Step 1: Install Plugin
1. Go to **Plugins** → **Add New**
2. Search for "Site Kit by Google"
3. Click **Install Now**
4. Click **Activate**

### Step 2: Set Up Site Kit
1. Click **Start Setup** in the Site Kit notice
2. Sign in with your Google account
3. Grant necessary permissions
4. Complete the setup wizard

## Configuration

### Google Analytics Setup
1. Go to **Site Kit** → **Settings**
2. Click **Connect Service** for Analytics
3. Select or create an Analytics property
4. Choose tracking configuration
5. Click **Configure Analytics**

### Google Search Console Setup
1. Go to **Site Kit** → **Settings**
2. Click **Connect Service** for Search Console
3. Verify site ownership
4. Select property
5. Complete setup

### Google AdSense Setup
1. Go to **Site Kit** → **Settings**
2. Click **Connect Service** for AdSense
3. Sign in with AdSense account
4. Link your AdSense account
5. Configure ad settings
