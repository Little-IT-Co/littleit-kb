# Plugins

WordPress plugins extend the functionality of your WordPress site. This section covers essential plugins and their configuration.

## Essential Plugins

### Recommended Plugins
- [Google Site Kit](/kb/wordpress/plugins/google-site-kit) - Google services integration
- [Elementor](/kb/wordpress/plugins/elementor) - Page builder
- [Limit Login Attempts](/kb/wordpress/plugins/limit-login-attempts) - Security plugin
- [WP Mail Options](/kb/wordpress/plugins/wp-mail-options) - Email configuration

## Installing Plugins

### From WordPress Dashboard
1. Go to **Plugins** → **Add New**
2. Search for the plugin name
3. Click **Install Now**
4. Click **Activate**

### Manual Installation
1. Download plugin ZIP file
2. Go to **Plugins** → **Add New**
3. Click **Upload Plugin**
4. Choose the ZIP file
5. Click **Install Now** then **Activate**

## Plugin Management

### Activating and Deactivating
- Activate plugins from the Plugins page
- Deactivate plugins when not in use
- Don't delete plugins you might need later

### Updating Plugins
- Update plugins regularly for security
- Check for updates on Plugins page
- Backup before updating
- Test after updates

## Best Practices

- **Keep Plugins Updated** - Regular updates include security patches
- **Minimize Plugin Count** - Only install necessary plugins
- **Read Reviews** - Check plugin ratings and reviews
- **Test First** - Test plugins on staging site
- **Remove Unused Plugins** - Delete plugins you don't use
