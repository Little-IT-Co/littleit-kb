# Installation Guides

WordPress can be installed on various platforms. This section covers installation methods for different hosting environments.

## Installation Methods

- [Installing on cPanel](/kb/wordpress/installation/cpanel) - Using cPanel's one-click installer
- [Installing on Docker](/kb/wordpress/installation/docker) - Containerized WordPress installation
- [Installing on Coolify](/kb/wordpress/installation/coolify) - Deploying with Coolify

## Prerequisites

### Requirements
- PHP 7.4 or higher
- MySQL 5.7 or higher (or MariaDB 10.3+)
- Apache or Nginx web server
- HTTPS support (recommended)

### Before Installation
- Have domain name ready
- Create database
- Obtain FTP/server access
- Have admin credentials ready

## Quick Installation Checklist

- [ ] Download WordPress
- [ ] Create database and user
- [ ] Upload WordPress files
- [ ] Configure wp-config.php
- [ ] Run installation wizard
- [ ] Complete initial setup
- [ ] Install security plugins
- [ ] Configure backups

## Manual Installation

### Step 1: Download WordPress
1. Go to [wordpress.org/download](https://wordpress.org/download)
2. Download latest WordPress
3. Extract ZIP file

### Step 2: Upload Files
1. Connect via FTP/SFTP
2. Upload files to web root
3. Ensure correct file permissions
4. Verify upload completion

### Step 3: Create Database
1. Access database management
2. Create new database
3. Create database user
4. Grant privileges

### Step 4: Configure
1. Rename wp-config-sample.php
2. Edit wp-config.php
3. Add database credentials
4. Set security keys

### Step 5: Install
1. Visit your domain
2. Run installation wizard
3. Enter site information
4. Create admin account

## Post-Installation

### Essential Steps
- Change default admin username
- Install security plugins
- Set up backups
- Configure permalinks
- Install essential plugins

### Security Setup
- Enable two-factor authentication
- Configure login limits
- Set up SSL certificate
- Review file permissions
