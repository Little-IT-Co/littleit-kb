# Google Drive, Docs, Sheets and Slides

Google Drive provides cloud storage, while Google Docs, Sheets, and Slides offer powerful productivity tools for creating and collaborating on documents, spreadsheets, and presentations.

## Overview

These integrated services allow you to store files securely in the cloud and work collaboratively on documents in real-time with your team.

## Google Drive

Google Drive is cloud storage that keeps your files safe and accessible from anywhere.

### Key Features
- **Cloud Storage** - Store files securely in the cloud
- **File Sharing** - Share files and folders with specific people or your entire organization
- **Version History** - View and restore previous versions of files
- **Offline Access** - Access files even without internet connection
- **Integration** - Works seamlessly with Docs, Sheets, Slides, and other Google services

### Common Tasks
- **Upload Files** - Drag and drop files into Drive or use the "New" button
- **Share Files** - Right-click a file and select "Share" to set permissions
- **Organize** - Create folders to organize your files
- **Search** - Use the search bar to quickly find files

## Google Docs

Google Docs is a word processor that lets you create and edit documents online.

### Key Features
- **Real-time Collaboration** - Multiple people can edit simultaneously
- **Comments and Suggestions** - Leave comments and suggest edits
- **Templates** - Use pre-made templates or create your own
- **Add-ons** - Extend functionality with add-ons
- **Export Options** - Download as PDF, Word, or other formats

### Common Tasks
- **Create Documents** - Click "New" → "Google Docs"
- **Format Text** - Use the formatting toolbar
- **Add Images** - Insert → Image
- **Share Documents** - Click the "Share" button

## Google Sheets

Google Sheets is a spreadsheet application for creating and managing data.

### Key Features
- **Real-time Collaboration** - Multiple users can edit simultaneously
- **Formulas and Functions** - Powerful calculation capabilities
- **Charts and Graphs** - Visualize your data
- **Conditional Formatting** - Highlight important data
- **Data Validation** - Control what data can be entered

### Common Tasks
- **Create Spreadsheets** - Click "New" → "Google Sheets"
- **Enter Data** - Click a cell and start typing
- **Use Formulas** - Start with "=" to create formulas
- **Create Charts** - Insert → Chart

## Google Slides

Google Slides is a presentation tool for creating and sharing presentations.

### Key Features
- **Real-time Collaboration** - Work together on presentations
- **Templates** - Professional presentation templates
- **Animations** - Add transitions and animations
- **Presenter View** - Present with speaker notes
- **Publishing** - Publish presentations online

### Common Tasks
- **Create Presentations** - Click "New" → "Google Slides"
- **Add Slides** - Click the "+" button or use shortcuts
- **Insert Media** - Add images, videos, and shapes
- **Present** - Click "Present" to start your presentation

## Collaboration Features

All these tools support:
- **Real-time Editing** - See changes as they happen
- **Comments** - Leave feedback and suggestions
- **Version History** - View and restore previous versions
- **Sharing Permissions** - Control who can view, comment, or edit
- **Notifications** - Get notified about changes and comments

## Best Practices

- Organize files in folders by project or team
- Use clear naming conventions for files
- Set appropriate sharing permissions
- Regularly review and clean up shared files
- Use comments for collaboration rather than editing directly
- Take advantage of templates for common documents
