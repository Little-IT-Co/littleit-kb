# Outlook & Exchange

Outlook and Exchange Online provide a robust email and calendaring solution for Microsoft 365 organizations.

## Overview

Exchange Online powers Outlook email, providing secure, reliable email delivery with advanced features for business communication and collaboration.

## Key Features

- **Professional Email** - Custom email addresses with your domain
- **Calendar** - Shared calendars and meeting scheduling
- **Contacts** - Centralized contact management
- **Tasks** - Task management and tracking
- **Security** - Advanced threat protection and spam filtering
- **Large Mailboxes** - Generous storage capacity
- **Mobile Access** - Access from any device
- **Web and Client Access** - Use Outlook on the web or desktop/mobile clients

## Accessing Outlook

You can access Outlook through:
- **Outlook on the Web** - Browser-based access at [outlook.office.com](https://outlook.office.com)
- **Outlook Desktop Client** - Full-featured desktop application
- **Mobile Apps** - Outlook app for iOS and Android
- **Email Clients** - Configure IMAP, POP3, or Exchange ActiveSync

## Common Tasks

### Sending Email
1. Click "New message" in Outlook
2. Enter recipients, subject, and message
3. Click "Send"

### Managing Calendar
- Create calendar events
- Share calendars with team members
- Schedule meetings with automatic room finder
- Set up recurring meetings

### Managing Contacts
- Add contacts manually or import from other sources
- Create contact groups for easy email distribution
- Sync contacts across devices

## Security Features

- **Advanced Threat Protection** - Protection against malicious links and attachments
- **Spam Filtering** - Built-in spam and phishing protection
- **Data Loss Prevention** - Policies to prevent sensitive data leakage
- **Encryption** - Email encryption options available
